# JU CRT Phase 1 – DSA Practice Repository

## Overview

This repository contains my solutions and practice programs completed during the **Campus Recruitment Training (CRT) – Phase 1** focused on **Data Structures and Algorithms (DSA)**.

The objective of this training phase was to strengthen problem-solving skills, improve coding proficiency in C++, and build a solid foundation in core DSA concepts frequently asked in technical interviews and placement assessments.

All solutions are organized day-wise according to the practice schedule followed during the CRT program.

---

## Topics Covered

### Day 1 – Arrays

* Insert Element in Array
* Delete Element from Array
* Second Largest Element

### Day 2 – Strings

* String Operations
* Palindrome Check
* Anagram Check
* Reverse String

### Day 3 – Searching Algorithms

* Binary Search (Iterative)
* Binary Search (Recursive)
* First and Last Occurrence of an Element

### Day 4 – Recursion & Sorting

* Merge Sort
* Recursion Basics
* Factorial Using Recursion
* Power Calculation
* Tower of Hanoi
* Array Search Problems

### Day 5 – Linked Lists

* Singly Linked List
* Doubly Linked List
* Linked List Operations

### Day 6 – Stacks

* Stack Implementation Using Array
* Stack STL
* Reverse a Stack
* Sort a Stack
* Insert at Bottom of Stack
* Next Smaller Element
* Previous Smaller Element
* Floyd's Cycle Detection Algorithm

### Day 7 – Queues & Deques

* Queue Implementation Using Array
* Queue STL
* Circular Queue
* Deque Implementation
* Reverse Queue
* Reverse Queue in K Groups

### Day 8 – Trees

* Introduction to Trees
* Basic Tree Operations
